## 说明

开始项目有一些HTML和CSS样式来显示Memory Game项目的静态版本。 您需要将此项目从静态项目转换为交互式项目。 这将需要修改HTML和CSS文件，但主要是JavaScript文件。

要开始，请打开 `js/app.js` 并开始构建应用程序的功能

有关具体的详细说明，请查看 [Udacity Classroom](https://classroom.udacity.com/me)中的项目说明。
